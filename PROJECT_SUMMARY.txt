## 🗄️ DATABASE SCHEMA

### Table: pesanan (Orders)
```sql
id                 INT AUTO_INCREMENT PRIMARY KEY
nama               VARCHAR(100) - Customer name
alamat             TEXT - Full address
nomor_hp           VARCHAR(15) - Phone number
jenis_kain         VARCHAR(50) - Fabric type
lingkar_dada       INT - Chest circumference
lingkar_pinggang   INT - Waist circumference
lingkar_bahu       INT - Shoulder width
panjang_lengan     INT - Sleeve length
lingkar_paha       INT - Thigh circumference
panjang_celana     INT - Pant length
panjang_baju       INT - Shirt length
harga_estimasi     DECIMAL(10,2) - Estimated price
catatan            TEXT - Additional notes
status             VARCHAR(20) - Order status
tanggal_pemesanan  TIMESTAMP - Order date
created_at         TIMESTAMP - Creation date
updated_at         TIMESTAMP - Last update date
```

### Table: harga_kain (Fabric Prices)
```sql
id                 INT AUTO_INCREMENT PRIMARY KEY
jenis_kain         VARCHAR(50) UNIQUE - Fabric type
harga_dasar        DECIMAL(10,2) - Base price
deskripsi          TEXT - Description
created_at         TIMESTAMP - Creation date
```

## 🚀 QUICK START

### 1. Setup Database (Menit 1)
```bash
1. Buka phpMyAdmin: http://localhost/phpmyadmin
2. Buat database: treadify
3. Import: db/treadify.sql
```

### 2. Verify Configuration (Menit 2)
```bash
Edit: includes/koneksi.php
Check: DB_HOST, DB_USER, DB_PASS, DB_NAME
```

### 3. Run Website (Menit 3)
```bash
1. Start XAMPP (Apache + MySQL)
2. Access: http://localhost/threadify2/
3. Website auto-redirect to beranda.php
```

**Total Setup Time: ~5 menit** ⏱️

## 🆘 TROUBLESHOOTING

### Database Connection Failed
**File**: `includes/koneksi.php`
**Check**: DB_HOST, DB_USER, DB_PASS, DB_NAME

### Form Not Submitting
**File**: `pages/pesan.php`
**Check**: Form ID, JavaScript console errors

### Dynamic Price Not Updating
**File**: `pages/pesan.php` (JavaScript section)
**Check**: Select element ID, event listeners

### Page Not Found
**Check**: Folder structure, file paths
**Verify**: `threadify2` folder di `htdocs`
